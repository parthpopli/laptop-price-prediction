
# Laptop Price Prediction

This project builds a machine learning regression model to predict laptop prices based on hardware specifications.

## Model Performance

Training R2 Score: 0.984
Test R2 Score: 0.807
MAE: 0.181
MSE: 0.073

## Key Insights

- RAM strongly influences laptop price
- SSD size also impacts pricing
- Larger screens correlate with heavier laptops
